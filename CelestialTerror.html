The SceneChangeManager

This is pretty self explanatory. These are all the variables the script needs to function properly

And then comes the actual brains of this entire script

What this does is basically change the rooms in the background making it look more smooth from the player perspective. First it checks if the ChangeState is not 'END'. and if it isn't it checks if the timer isn't smaller than 0 and the change state is stil not 'END'. Then everything checks out to be fine. It sets the timer back to 0. It turns off the SceneChange GameObject.And at the end puts the ChangeState to 'END'.

Then in the same script the second part if the timer is bigger than 0 but still smaller than 1 AND the ChangeState is not 'MIDDLE'. It searched for the ROoms GameObject and makes a local variable named 'all'. After that it sets 2 more local variables named FromRoomL and targetRoom and sets them to fromRoom and toRoom. After finding the targetRoom it sets it active and turns the FromRoomL off. and after that it sets the ChangeState to 'MIDDLE'.
